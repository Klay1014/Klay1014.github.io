ETF分析

0050 配息資料自動抓取與整理

最近在看 0050 的配息規律，想知道過去幾年到底是怎麼配的。 雖然元大官網查得到，但每次點年份、月份實在太麻煩，乾脆自己寫個小工具把它一次抓下來。 專案目標 針對元大 0050 這支 ETF，抓取歷年所有配息紀錄 把資料整理成 .csv，方便日後用 Excel / R / Python 做分析 練習 API 觀察技巧，理解官網背後怎麼存資料的 觀察過程 元大的配息查詢網頁是 Nuxt 打造的 SPA（單頁應用程式），一開始還以為要用 Selenium 操作 UI，但實際看一下 Chrome DevTools 的 Network tab，發現它會打到一個後端 API，資料是用 JSON 回傳。 而且一次就可以查詢 2005～2025 的資料，根本不用手動分頁。 最後抓到的欄位有： 基金名稱：都是「元大台灣卓越50基金」 配息頻率：大部分是半年配息 每單位配息金額 當期配息率 除息日 發放日 資料筆數一共 29 筆，橫跨 2005～2025，內容完整，跟網站上對得起來。 額外筆記 JSON 結構有點詭異，Data 裡還包了一個 Data，要兩層解包才拿得到 list。 存成 csv 時記得用 utf-8-sig，這樣在 R 或 Excel 開才不會亂碼。 RStudio 的 Import Text Data 可以直接吃，不用特別轉格式。 如果之後想查 0056、00878 等等，只要改 FundId 就行，API 是共用的。 下一步想做的 之後可以畫一下配息趨勢圖，看它是穩定往上、還是時高時低 也可以試著加總年度配息金額、算一下殖利率變化 或是找個 baseline（比如台積電）比較一下哪個現金流比較穩 ...
